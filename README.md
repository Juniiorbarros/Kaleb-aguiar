# Cliente Atelier Kaleb Aguiar

App React/Vite para vitrine premium e linkpage premium.

Rotas:

- `/` vitrine cinematografica.
- `/links` linkpage premium para Instagram, WhatsApp, categorias e localizacao.

Comandos:

```bash
npm install
npm run dev
npm run build
```

A configuracao principal fica em `src/config/client.config.js`. Novas fotos devem entrar primeiro em `references/user-provided/<batch>/raw/`, depois serem curadas para `public/assets/kaleb-aguiar/` quando forem usadas no app.
